Contable Monorepo

Monorepo scaffold para una aplicación contable con frontend (React + Vite), API (Fastify + Prisma) e infraestructura de ejemplo.

Estructura generada automáticamente. Completar dependencias y detalles por proyecto.

Para empezar:

- Instalar dependencias con `pnpm install` en la raíz (si usas pnpm).
- Abrir `apps/web` y `apps/api` para completar `package.json` y dependencias.
